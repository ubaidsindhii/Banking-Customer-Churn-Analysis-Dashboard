# Banking Customer Churn Analysis Dashboard

This project focuses on analyzing customer churn behavior in the banking sector using Power BI. The dashboard provides actionable insights into customer retention, high-risk segments, geography-based churn trends, demographic analysis, and forecasting.

The objective of the project is to help businesses identify factors contributing to customer churn and support data-driven retention strategies.

Technologies & Tools Used: Power BI, DAX, Power Query, Microsoft Excel/CSV, Data Visualization, KPI Reporting, Forecasting, Customer Segmentation, and Business Analytics techniques.
